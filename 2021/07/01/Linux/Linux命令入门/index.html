Linux命令入门

终端命令格式

command [-options] [parameter]

命令名 选项 参数

查阅帮助信息：

command —help

man command

文件目录常用命令

  1. ls

    • -a 所有子目录与文件、包括隐藏的
    • -l 以列表形式显示详细信息
    • -h 配合-l 人性化的显示文件大小

  2. cd

  3. touch

  4. mkdir

    • -p 递归创建目录

  5. rm

    • -f 强制删除，忽略不存在的文件
    • -r 递归删除目录下的内容，删除文件夹时添加

  6. tree [目录名]

    ​ 以树状列出文件目录结构

    • -d 只显示目录

  7. cp 源文件 目标文件

    • -i 覆盖文件前提示
    • -r 若是目录文件，递归复制目录下的所有子目录和文件

  8. mv 源文件 目标文件

    • -i 覆盖文件前提示

  9. cat 文件名

    • -b 对非空输出行编号
    • -n 对输出的所有行编号

  10. more 文件名

    分屏显示文件内容

    • 空格 显示下一屏
    • enter 滚动一行
    • b 回滚一屏
    • f 前滚一屏
    • q 退出
    • /word 搜索word字符串

  11. grep 搜索文本 文件名

    文本搜索工具

    • -n 显示匹配行以及行号
    • -v 显示不包含匹配文本的所有行（求反）
    • -i 忽略大小写

    常用两种查找模式

    • ^a 搜索以a开头的行
    • ke$ 搜索以ke结束的行

  12. echo 文字内容

    显示指定的文字，通常和重定向一起使用

  13. 重定向 >, >>

    ‘>’ 表示输出，会覆盖原有内容

    ‘>>’ 表示追加，把内容追加到文件末尾

  14. 管道|

    一个命令的输出通过管道作为另一个命令的输入

远程管理常用命令

  1. shutdown

    关机、重启

    • -r 重新启动
    • -c 取消之前指定的关机计划

    不指定选项和参数，默认1分钟后关闭电脑

    shutdown now 表示现在

    shutdown 20:25 表示在20:25关机

    shutdown +10 表示10分钟后关机

    shutdown -c 取消之前指定的关机计划

  2. ifconfig

    查看计算机当前网卡信息

  3. ssh 用户名@ip
