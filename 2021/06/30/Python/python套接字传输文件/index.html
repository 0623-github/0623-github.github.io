python套接字传输文件

实现功能

  1. 查看文件夹下的文件
  2. 下载文件
  3. 上传文件

只有服务端代码，用的TCP测试工具进行测试

传输数据格式

  1. 查看文件：

    • 客户端发往服务端：1:filepath
    • 服务端发往客户端：文件个数:文件类型:文件名字|文件类型:文件名字|文件类型:文件名字|……
    • 冒号当分隔符用

  2. 下载文件：

    • 客户端发往服务端：2:filepath

    • 服务端发往客户端：文件内容结尾标志

  3. 上传：

    • 客户端发往服务端：3:filename:文件内容
